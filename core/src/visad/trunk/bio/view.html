The View Tab

The View tab contains functions for altering how data gets displayed, and for manipulating those displays.

The 2-D display

VisBio has a 2-D display for showing the current slice at the current timestep. If arbitrary slicing is enabled, the 2-D display shows the current arbitrary data slice.

To toggle the visibility of the 2-D display, check or uncheck the "2-D" checkbox. Use the Zoom and Reset buttons to the right of the checkbox for adjusting the display's zoom factor. Alternately, shift + left mouse drag along the 2-D display also adjusts zoom. Lastly, left mouse drag translates the data within the display.

The 3-D display

VisBio has a 3-D display for showing the stack of slices at the current timestep.
